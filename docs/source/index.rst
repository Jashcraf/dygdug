dygdug
======

*dig that dark hole.*

**dygdug** is a toolkit for coronagraph modeling, propagation, and
focal-plane wavefront control for high-contrast imaging.

The documentation is organized along the `Diátaxis
<https://diataxis.fr/>`_ framework:

- **Tutorials** — learning-oriented lessons to get you started.
- **How-to guides** — task-oriented recipes for specific problems.
- **Explanations** — discussion of the concepts and design behind dygdug.
- **API reference** — the generated reference for every public module.
